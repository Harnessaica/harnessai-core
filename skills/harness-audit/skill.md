# harness-audit

Run the HarnessAI AI Audit workflow with a client.

## When to Use
During or after a 90-minute AI Audit session with a client.

## Inputs Needed
- Client business type and size
- Top 3 time-consuming manual tasks they described
- Any existing tools they use (software, spreadsheets, etc.)
- Their comfort level with technology

## Output
A written AI Audit action plan: top 3 AI opportunities, specific tools recommended, rough implementation effort for each.
